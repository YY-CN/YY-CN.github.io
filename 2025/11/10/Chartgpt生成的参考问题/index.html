综合测试：题目与参考答案（Markdown 版）

环境按题面用 Python ≥3.8、PyTorch ≥2.0。

一、目录与通用脚手架

utils.py（可复用）

二、Q1 面向对象（类/继承/接口/构造）

要求：展示继承、抽象接口（Python 用 abc）、构造函数与多态。

三、Q2 感知器 / 激活 / 交叉熵 / 逻辑回归（纯 Python/Numpy）

建议：感知器已在 Q1。再补 sigmoid、ReLU、交叉熵、逻辑回归训练。

四、Q3 PyTorch2.0 CNN（按图或描述写类）

最常用写法：Conv → BN → ReLU → Pool + 头部 GAP → Linear。把提供的模型图参数填进模板的通道数、kernel、stride 即可。

若图含残差/分支，写一个 ResidualBlock 或在 forward 里显式 y = x1 + x2 / torch.cat([...], dim=1)。

五、Q4 PyTorch2.0 LSTM（按图或描述写类）

常见：Embedding → LSTM → (可选) Dropout → Linear。

六、Q5 Transformer + （LSTM 或 CNN）综合

给一个最小可用混合：TransformerEncoder → LSTM → 线性分类。把层数/头数按题面图填。

七、提交小贴士（省坑）

  • 固定随机种子；打印 参数量/每层 shape 做自检。
  • 题3/4/5 都按提供的模型图或描述把参数填进去（通道数、kernel、stride、层数、头数…）。
  • 训练可用极简 loop（小 batch 跑几步即可证明可运行）；若无数据就造 toy data。
