题3（20分）—— CNN 图像分类（MNIST）

要求：

  • 输入：(batch_size, 1, 28, 28)
  • 两层卷积 + 一层全连接
  • 用 ReLU
  • 输出：(batch_size, 10)

1. 标准答案代码

2. 详细讲解

  1. 卷积层尺寸推导：
    • 输入：(1, 28, 28)
    • conv1：padding=1, kernel_size=3, stride=1 → 高宽仍为 28
      • 输出 (32, 28, 28)
    • 池化 MaxPool2d(2, 2) → 高宽减半：28→14
      • 输出 (32, 14, 14)
    • conv2：同理 → (64, 14, 14)
    • 再池化 → (64, 7, 7)
  2. 全连接层输入维度：
    • 每个样本展平后长度： 64 * 7 * 7
    • 所以 nn.Linear(64*7*7, 10)
  3. ReLU：
    • F.relu(x) 是常见激活函数，保持正数不变，负数变为 0，增加非线性表达能力。

模型 TextLSTM：

  • 构造：单层 LSTM + Linear
  • 输出：(batch, num_classes)

1. 标准答案代码

2. 详细讲解

  1. nn.LSTM 的关键参数：
    • input_size=embed_dim：每个时间步输入向量维度
    • hidden_size=hidden_dim：隐藏状态维度
    • batch_first=True：张量形状是 (batch, seq_len, feature)
  2. lstm(x) 返回：
    • output: 所有时间步的输出，形状 (batch, seq_len, hidden_dim)
    • (h_n, c_n): 最后一个时间步的隐藏状态和值，形状 (num_layers * num_directions, batch, hidden_dim)
  3. 分类常用做法：
    • 取 h_n[-1] 作为句子表示（最后一层、最后一个方向的隐藏状态）
    • 传入 Linear 变成 num_classes 维的 logits

  • 输入 (seq_len, batch_size, d_model)，输出同形状

1. 标准答案代码

2. 详细讲解

  1. MultiheadAttention 输入输出形状：

    • 这里设定 batch_first=False，所以输入/输出都是 (seq_len, batch, d_model)
    • self.self_attn(src, src, src) 表示自注意力：
      • query = key = value = src

  2. 残差连接：

    • 原始输入 src 加上注意力输出 src2
    • 再做 LayerNorm，稳定训练

  3. 前馈网络（FFN）：

    • 维度变换：d_model → dim_ff → d_model
    • 中间用 ReLU 激活

  4. 第二个残差：

    • FFN 输出加回原来的 src 再做 LayerNorm
